(async, static) getUsrPwd(req, res) → {Promise.<void>}

Procesa el registro de un nuevo usuario, validando los datos de entrada, hasheando la contraseña y guardando la información del usuario en la base de datos.
Parameters:
Name Type Description
req Object Objeto de solicitud (Request) de Express, que contiene los datos de usuario y contraseña.
Properties
Name Type Description
body Object Cuerpo de la solicitud con los datos del usuario.
Properties
Name Type Description
username string Nombre de usuario.
passwordUser string Contraseña sin hashear.
cookie string Token JWT que contiene el correo electrónico del usuario.
res Object Objeto de respuesta (Response) de Express.
